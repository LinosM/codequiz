# UNC Bootcamp Homework 4 - Code Quiz

## Overview

This quiz asks the user questions relating to Javascript. A timer of 75 seconds will start once the quiz begins and deducts 10 seconds whenever the user gets a question wrong. At the end the user's score is displayed along with a text box to enter their initials.
